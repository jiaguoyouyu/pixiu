# Pixiu Partnership Catalyst Report Generator

This workflow turns the Major Partnership Catalyst Radar skill into a repeatable local report generator.

## Command

python3 scripts/generate_partnership_catalyst_report.py --input templates/major_partnership_catalyst_input_template.md

## Output

Generated reports are written to:

outputs/partnership_catalyst_reports/

Generated reports are local outputs and should not be tracked by Git.
